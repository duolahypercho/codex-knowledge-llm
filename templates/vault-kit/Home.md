# Knowledge Vault

This vault is designed as a feedback loop, not a filing cabinet.

## Core Areas

- [[inbox/Inbox]] - raw captures, daily briefs, and session summaries.
- [[notes/Notes]] - source material and source index notes.
- [[ideas/Ideas]] - synthesis, teardowns, implementation notes, and original thinking.
- [[projects/Projects]] - active work and decisions.
- [[Prompts - Daily And Weekly Synthesis]] - daily and weekly output loops.
- [[Automation - Capture Pipeline]] - capture and routing blueprint.

## Daily Loop

1. Capture without organizing.
2. Let new material land in `inbox` or `notes`.
3. Ask Codex for a daily session summary.
4. Move only useful raw captures into `notes`, `ideas`, or `projects`.

## Weekly Loop

1. Update [[AGENTS]].
2. Run the weekly synthesis prompt.
3. Turn the best contradiction, gap, or thesis into one concrete project move.
