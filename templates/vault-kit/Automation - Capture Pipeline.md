# Automation - Capture Pipeline

Created by: {{OWNER}}

## Flow

Capture tools -> import workflow -> Obsidian vault -> Codex synthesis.

## Supported Inputs

- X posts and articles.
- Web articles and newsletters.
- YouTube or podcast transcripts.
- Books, Kindle highlights, and reading notes.
- PDFs as extracted text.
- Readwise exports.
- n8n or Telegram captures.

## Routing

- Complete sources go to `notes`.
- Original thinking and synthesis go to `ideas`.
- Raw captures go to `inbox`.
- Active work goes to `projects`.

## Rule

When unsure, put the raw capture in `inbox` and let Codex route it later.
