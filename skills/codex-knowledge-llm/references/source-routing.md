# Source Routing

Use these routes after cleaning obvious UI junk such as keyboard shortcut text, timestamps unrelated to the content, repeated app labels, and social UI controls.

## Persuasive Posts And Articles

Use for X articles, long posts, essays, newsletters, and landing-page-style writing that tries to persuade, teach a market thesis, or sell a behavior.

Create:
- `notes/<Title> - Original.md`
- `notes/<Title> - Index.md`
- `ideas/<Title> - Structure Teardown.md`

The teardown should explain:
- High-level structure.
- Why the post works.
- Psychological patterns.
- Content needed from the vault owner to recreate it.
- Rebuild template.
- Rewrite prompt.

## Educational Tutorials And Technical Guides

Use for implementation walkthroughs, coding tutorials, operating procedures, and setup guides.

Create:
- `notes/<Title> - Original.md`
- `notes/<Title> - Index.md`
- `ideas/<Title> - Implementation Notes.md`

Implementation notes should include prerequisites, setup sequence, reusable commands, risks, edge cases, and next actions.

## YouTube, Podcast, And Transcript Sources

Use for video/audio transcripts, talks, interviews, and podcast clips.

Create:
- `notes/<Title> - Original.md`
- `notes/<Title> - Index.md`
- `ideas/<Title> - Concept Synthesis.md`

Concept synthesis should include main thesis, key claims, examples, memorable lines, disagreements/questions, and reusable ideas.

## Books, Kindle Highlights, And Reading Notes

Use for book notes, reading excerpts, Kindle exports, and longform study material.

Create:
- `notes/<Title> - Original.md`
- `notes/<Title> - Index.md`
- `ideas/<Title> - Book Notes.md`

Book notes should include core thesis, chapter/section themes, strongest ideas, useful frameworks, open questions, and applications.

## Raw Captures And Automation Imports

Use for Telegram messages, n8n imports, Readwise fragments, quick ideas, and rough inbox material.

Default:
- Create `inbox/YYYY-MM-DD-quick-capture.md`.

If the capture contains a complete source, route it using the source categories above.
