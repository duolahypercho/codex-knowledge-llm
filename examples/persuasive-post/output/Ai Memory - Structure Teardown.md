## One-Line Pattern

Name a common behavior, reveal the missed leverage, then offer a simple operating shift.

## High-Level Structure

1. Call out the default habit: using AI like a search box.
2. Name the cost: useful conversations disappear.
3. Introduce the better model: compounding context.
4. Make the model concrete with a short sequence.
5. Close with a memorable contrast: starting from zero versus accumulated context.

## Why It Works

The post works because it reframes a familiar tool without needing technical detail. It gives the reader a practical identity shift: from asking isolated questions to building an AI-readable knowledge system.

## Psychological Pattern

### Default Versus Upgrade

Pattern: "Most people do X. The leverage is Y. Here is the simple shift."

Why it works: Readers recognize the default behavior before being invited into a higher-agency workflow.

Use when: You want to launch a tool or workflow that changes how people use something they already know.
