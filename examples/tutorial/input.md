Source type: implementation tutorial

Title: Set Up A Local Knowledge Vault

Text:

Create a folder for your vault. Add subfolders for inbox, notes, ideas, and projects.

Put an AGENTS.md file at the root with your goals, current projects, and instructions for how the assistant should use the vault.

When adding source material, keep the original text separate from your synthesis. Link both from an index note so the vault stays easy to navigate.

Review the inbox once a day and move useful captures into notes or ideas.
