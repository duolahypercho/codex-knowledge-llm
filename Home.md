# Codex Knowledge LLM Vault

Created by: ziwenxu

This vault is the working knowledge base for the Codex Knowledge LLM plugin. It tracks the product, examples, implementation notes, launch positioning, and source material behind the repo.

## Core Areas

- [[inbox/Inbox]] - raw captures, daily briefs, and session summaries.
- [[notes/Notes]] - original source material and source index notes.
- [[ideas/Ideas]] - synthesis, teardowns, implementation notes, and original thinking.
- [[projects/Projects]] - active work and decisions.
- [[Prompts - Daily And Weekly Synthesis]] - daily and weekly output loops.
- [[Automation - Capture Pipeline]] - capture and routing blueprint.
- [[Graphify - Optional Knowledge Graph]] - optional graph layer for relationship mapping.

## Product Context

- Plugin: `codex-knowledge-llm`
- Goal: turn source material into an Obsidian-ready knowledge system that Codex can use across sessions.
- Public promise: install once, open Codex in your vault, paste any source, and get structured notes that compound over time.

## Daily Loop

1. Capture without over-organizing.
2. Let new material land in `inbox` or `notes`.
3. Ask Codex for a daily session summary.
4. Promote useful captures into `notes`, `ideas`, or `projects`.

## Weekly Loop

1. Update [[AGENTS]] when the repo direction changes.
2. Run the weekly synthesis prompt.
3. Turn the best contradiction, gap, or thesis into one concrete project move.
